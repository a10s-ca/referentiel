# Propriété des classes

Les propriétés associées à chacune des classes principales et secondaires du référentiel sont présentées dans la section Références.

Pour chaque propriété, les éléments suivants sont documentés :

* le nom de la propriété ;
* sa description, et parfois des conseils d’usage ou des exemples ;
* le caractère obligatoire, ou pas, de la propriété, dans le contexte de la classe concernée ;
* la cardinalité de la propriété ;
* le type ou format de données attendu.

Les valeurs possibles pour la cardinalité sont :

* 1 : obligatoire et unique
* 0..1 : optionnel et unique
* 1..N : au moins une valeur obligatoire, avec possibilité d’en énumérer plusieurs
* 0..N : aucune valeur obligatoire, avec possibilité d’en énumérer plusieurs

En ce qui concerne le dernier élément, les types de données sont présentés dans la section Références. Des explications portant sur l’utilisation des vocabulaires contrôlés sont également fournies dans cette section.
