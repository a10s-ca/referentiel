    Ceci est une classe utilitaire

    Pour faciliter la lecture, certaines propriétés communes à plusieurs classes principales ont été regroupées dans des classes utilitaires. Ces classes ne font donc pas directement partie du modèle.

    Propriété Description Priorité Cardinalité Type
    unité Numéro d’unité, bureau ou appartement

    Note: cette propriété et les propriétés suivantes sont inspirées des directrices sur l'adressage de Postes Canada.    		 Optionnel 1 Texte court
    numéro Numéro municipal Obligatoire 1 Texte court
    type de rue Type de rue Obligatoire 1 Abréviation de type de rue recommandée par Postes Canada
    rue Nom de la rue Obligatoire 1 Texte court
    ville Ville Obligatoire 1 Texte court
    direction de rue Direction de la rue (points cardinaux) Optionnel Abréviation des points cardinaux de Postes Canada
    province Province Obligatoire 1 Abréviation des noms de provinces recommandés par Postes Canada
    pays Pays Obligatoire 1 Code de pays à 3 caractères selon le standard ISO 3166-1
    code postal Code postal, en majuscules. Séparer les trois premiers caractères du code postal des trois derniers. On ne doit pas utiliser le trait d’union. Obligatoire 1 Texte court
